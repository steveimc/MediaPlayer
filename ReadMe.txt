MediaPlayer by Steve Macias

https://github.com/steveimc/MediaPlayer.git

Download the project folder from the git repository, place the folder in a web server
or if you have any Apache distribution environment like xampp place the project inside the htdocs folder.

Open the web app in a browser from the host address.

The app includes 4 song to play, and has the choice to upload a file.
Works in mobile and desktop.
Controls include skip previous, rewind, play, pause, fast forward, skip next.
You can also click on the time track to skip to that position in the song.
